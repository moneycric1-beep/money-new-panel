## Deploy Now (2 Minutes)

### Step 1: Push to GitHub
```bash
cd "money new panel"
git init
git add .
git commit -m "Money Panel"
git push origin main
```

### Step 2: Deploy on Railway
1. Go to [railway.app](https://railway.app)
2. Click "New Project"
3. Select "Deploy from GitHub repo"
4. Choose your repo
5. Done! Railway auto-deploys

### Step 3: Add Bot Variables
Railway Dashboard → Variables tab → Add:
```
TELEGRAM_BOT_TOKEN=your_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

### Step 4: Get URL & Test
- Railway gives you: `your-app.railway.app`
- Open URL → Upload APK → Check Telegram!

## Get Telegram Credentials

**Bot Token:**
1. Telegram → Search `@BotFather`
2. Send `/newbot`
3. Copy token

**Chat ID:**
1. Telegram → Search `@userinfobot`
2. Start chat → Get ID
